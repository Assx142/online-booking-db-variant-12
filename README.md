## 📋 Описание предметной области

Система позволяет клиентам записываться на банковские услуги: открытие вклада, консультацию, оформление кредита. Учитываются отделы банка, должности сотрудников, кредитная история клиента. Клиенты с низким кредитным рейтингом не могут записаться на услуги с крупными суммами вклада.

## 🗄️ Структура базы данных

Основные сущности:
- `clients` — клиенты банка с паспортными данными
- `credit_histories` — кредитные истории с рейтингом 300-850
- `services` — банковские услуги с минимальной суммой вклада
- `employees` — сотрудники с привязкой к отделам и должностям
- `appointments` — записи на обслуживание с проверкой бизнес-правил
